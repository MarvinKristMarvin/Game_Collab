visiteur peut :
- accèder à toutes les pages
- voir les profils (description, jobs, languages, rémunérations, nom, age, lien portfolio, adresse mail)
- s'inscrire
- se connecter

utilisateur connecté peut :
- modifier son profil
- voir les profils
- aller au suivant ou au précédant profil
- filtrer sa recherche de profils (jobs, rémunérations, languages, age, mots clés)
- se déconnecter
- se désinscrire

admin peut :
- supprimer profil
